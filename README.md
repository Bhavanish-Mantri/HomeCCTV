# HomeCCTV Intrusion Detection System

This is a Python-based CCTV Intrusion Detection System built on Kali Linux. It detects motion, captures evidence, and alerts via Telegram.

##  Features
- Real-time motion detection using OpenCV
- Video capture and frame storage
- Telegram alert integration
- Web dashboard using Flask
- Mobile IP webcam support

##  Technologies Used
- Python
- OpenCV
- Flask
- ffmpeg
- Telegram Bot API

##  Project Structure
- app.py — main application
- static/ — CSS, JS, images
- templates/ — HTML files
- intrusions/ — saved intrusion clips and logs

##  Note
Only for educational and ethical use. Do not use this for spying or unauthorized monitoring.
